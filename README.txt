# django-extend-auth

Это проект расширяет базовую модель User в django. 
Теперь можно добавлять свои поля в модель. В документации написано, что предпочтительно в каждом проекте делать так, 
потому что в работающем проекте (после команд python manage.py makemigrations и python manage.py migrate) 
расширить базовую модель довольно сложно.
